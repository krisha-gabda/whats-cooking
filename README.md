# What's Cooking

A full-stack restaurant discovery and review platform with social networking features, built as a formally assessed university project at the University of Manchester.

## My Role

Co-led a cross-functional team of 7. Personally responsible for:
- System architecture and technical decision-making across the full stack
- Relational database schema design — normalised tables with indexed foreign keys for efficient sorting and filtering by category and rating
- JWT authentication system — protected API endpoints, role-based access control for admin features, and secure session handling
- Social graph features — username-based friend search and a mutual friends recommendation algorithm
- Presented architecture decisions and live product to a panel of professors under formal evaluation

## Tech Stack

| Layer | Technologies |
|---|---|
| Frontend | React.js, HTML, CSS |
| Backend | PHP, Node.js, RESTful APIs |
| Database | MySQL |
| Auth | JWT, bcrypt |
| Version Control | Git, GitLab |

## Features

- Restaurant discovery with filtering by category and rating
- User reviews and star ratings
- JWT-secured user authentication and admin access control
- Friend search by username
- Mutual friends recommendation system
- Responsive UI
